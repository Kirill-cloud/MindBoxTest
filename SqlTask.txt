допустим у нас есть таблицы:
товары - Products 
категории - Categories
Для организации связи многие ко многим нужна ещё одна таблица 
Products2Categories, в  которой есть поля: 
    1) id_Product - ссылка на товар
    2) id_Category - ссылка на категорию

Для того чтобы выбрать все пары товар-категория, причем товар без категории
тоже должен быть выведен, можно использовать следующий запрос:

Select p.*,c.* from Products p -- p.*,c.*  используется чтобы вывести 
-- только информацию по товару/категории без данных связующей таблицы
left join Products2Categories p2c on p.id = id_Product
left join Categories c on c.id = p2c.id_Category
